However, implementation and structure vary from site to site, dev to dev. Let's spend a little time now building some quick consensus around structure for this small, initial set of paragraphs. (Example: for Left/Right Images do we use a field to allow the user to select orientation? Should accordions be made up of nested paragraph types?)
